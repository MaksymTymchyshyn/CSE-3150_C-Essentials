# Lab Week 1: Prefix Sums

## Functions
- `non_neg_prefix_sum` - Checks if all prefix sums stay >= 0
- `non_pos_prefix_sum` - Checks if all prefix sums stay <= 0

## Files
- `prefix_sums.h` - Function declarations
- `prefix_sums.cpp` - Implementation
- `test_prefix_sums.cpp` - Unit tests with DOCTEST

## How to Compile and Run

```bash
g++ -std=c++11 test_prefix_sums.cpp prefix_sums.cpp -o test_prefix_sums
./test_prefix_sums
```
