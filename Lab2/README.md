# Lab Week 2: Well-Balanced Lists

## How to Build and Run

### Build and Run Unit Tests

```bash
g++ -std=c++11 -o test_balanced_lists test_balanced_lists.cpp balanced_lists.cpp
./test_balanced_lists
```

### Build and Run Main Program

```bash
g++ -std=c++11 -o balanced_lists main.cpp balanced_lists.cpp
./balanced_lists
```
