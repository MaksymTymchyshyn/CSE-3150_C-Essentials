# Lab Week 4: Hybrid Vehicles - Multi-Inheritance

## How to Build and Run

### Build and Run Unit Tests

```bash
g++ -std=c++11 -o test_hybrid test_hybrid.cpp Vehicle.cpp GasolineVehicle.cpp ElectricVehicle.cpp HybridVehicle.cpp && ./test_hybrid
```

### Build and Run Main Program

```bash
g++ -std=c++11 -o main main.cpp Vehicle.cpp GasolineVehicle.cpp ElectricVehicle.cpp HybridVehicle.cpp
./main
```
